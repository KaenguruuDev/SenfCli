# SenfCLI

SenfCLI is the command-line interface for Senf, a secure backend service for managing encrypted environment files with SSH key-based authentication. It lets users initialize projects, manage profiles and keys, and sync encrypted `.env` files for secure sharing and version control of sensitive configuration data. It supports multiple SSH keys per user for flexible access management across devices and teams.

For more information, visit the [documentation](https://senf-docs.kaenguruu.dev).
